## Introduction

Pet Store is a E-commerce application that provides various shipping services and features. This repository contains the source code for the Pet Store application.

## Features

- *Login*: Login and manage accounts.
- *Create new Pet, Store and User *: Create new Pet, Store and User with Radnom Data.
- *Update Pet, Store and User *: Update Pet, Store and User with Radnom Data.
- *Delete Pet, Store and User*: Ensure Pet, Store and User is Deleted.

## Installation

To set up the Pet Store application locally, follow these steps:

1. Clone the repository:
*   git clone https://github.com/MennaEesa/Pet_Store_API
2. Navigate to the project directory:
*   cd Pets Store
3. Install dependencies:
*   npm install
4. Configure the application:

* Update configuration files with necessary settings.
5. Run the application:
*   npm start
